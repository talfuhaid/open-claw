## Notification Delivery

To notify the user, run:
`notify_main` tool with the alert text.

Do NOT use sessions_send. Use this script instead.
CRITICAL: Do NOT write the notification as your reply text.
Your heartbeat target is "none" — reply text goes nowhere.
You MUST use the `notify_main` script to deliver notifications.
Do NOT send "Done" or any confirmation — only important email notifications.
